# telco-churn-analysis
telco customer churn is a dataset about customer telecommunication service data. Telco Churn Analysis have purpose to gain information on what customer behavior toward telecommunication service.This analysis can be a reference to telco company for more understanding on how they should do to improve their service. 

5 question will be answered in this analysis
1. how diverge does customer on service choice?
2. is there any factore on why customer chose their service?
3. what reason customer doesnt have interest to chose some service?
4. what customer segment that tend to chose certain service?
5. what telco company need to do to improve or maintain their service interest?
